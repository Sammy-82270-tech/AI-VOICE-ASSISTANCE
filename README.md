# AI Voice Assistant

AI Voice Assistant is a Python-based intelligent assistant that allows users to interact with a computer using voice commands. The system uses Speech Recognition and Natural Language Processing (NLP) to understand user input and respond with voice output.

The assistant can perform multiple tasks such as:
- Playing music
- Opening websites
- Fetching real-time news
- Answering queries using Google Gemini API
- Basic computer automation

The project demonstrates how Artificial Intelligence can enable hands-free interaction between humans and machines, making digital systems more accessible and efficient.

## Technologies Used
- Python
- SpeechRecognition
- gTTS (Google Text-to-Speech)
- Pygame
- Requests
- Google Gemini API

## Working Process
1. The system listens to the user's voice command.
2. Speech Recognition converts voice into text.
3. NLP analyzes the command.
4. The system executes predefined tasks or queries the AI model.
5. The response is converted into speech and played back to the user.

This project highlights the potential of conversational AI in fields like education, automation, smart homes, and personal productivity.
